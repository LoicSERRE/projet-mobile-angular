Informations

Cette application permet à vous, parents, de suivre l'acutalité de votre enfant dans son école.

Vous pouvez swipe vers la gauche pour afficher les instructions suivante

Articles

Vous pouvez consulter les articles publiés par l'école, comme leur date, leur importance

Vous avez un menu au haut a gauche, vous pouvez soit cliquer sur les 3 barres horizontale soit swipe a partir du bord gauche

VOus pouvez cliquer sur articles pour afficher les articles publiés par l'école

Galerie d'images

Vous pouvez consulter les images publiées par l'école, comme leur date, leur importance

Vous avez un menu au haut a gauche, vous pouvez soit cliquer sur les 3 barres horizontale soit swipe a partir du bord gauche

Vous pouvez cliquer sur Galerie d'images pour afficher les images publiées par l'école

Dates importantes

Vous pouvez consulter les dates importantes de l'école

Vous avez un menu au haut a gauche, vous pouvez soit cliquer sur les 3 barres horizontale soit swipe a partir du bord gauche

Vous pouvez cliquer sur Dates importantes pour afficher les dates importantes de l'école

Informations de contact

Vous pouvez consulter les informations de contact de l'école

Vous avez un menu au haut a gauche, vous pouvez soit cliquer sur les 3 barres horizontale soit swipe a partir du bord gauche

Vous pouvez cliquer sur Informations de contact pour afficher les informations de contact de l'école

Sur cette pages vous avez 3 boutons, un pour appeler l'école, un pour envoyer un mail et un pour afficher l'adresse de l'école sur google maps

Favoris

Vous pouvez consulter les articles et images que vous avez mis en favoris

Vous avez un menu au haut a gauche, vous pouvez soit cliquer sur les 3 barres horizontale soit swipe a partir du bord gauche

Vous pouvez cliquer sur Favoris pour afficher les articles et images que vous avez mis en favoris
